The Pipeline

How CogniLock Works

From raw brainwaves to secure authentication in four elegant steps.

01

EEG Signal Capture

Record brainwave data from 4 electrode channels — P4 (parietal), Cz (central), F8 (frontal), and T7 (temporal).

4 Channels Non-invasive Standard 10-20 System
02

Signal Preprocessing

Raw EEG signals are normalized using StandardScaler and segmented into overlapping windows of 100 samples with a step size of 10.

StandardScaler Sliding Window Overlap Chunking
03

Deep Learning Classification

A 1D Convolutional Neural Network with 3 conv blocks (64→128→256 filters), batch normalization, and dropout processes each chunk independently.

1D CNN BatchNorm Softmax Output
04

Majority Vote & Decision

Predictions across all chunks are aggregated via majority voting. Access is granted only if the average confidence exceeds the security threshold.

Majority Voting Confidence Threshold Access Control
